# BOB's Burgers - KOT

A simple `Kitchen Order Tickets (KOT)` system built to demonstrate the usage `mySQL`, `Node.js`, `Express`, `Handlebars` and a homemade `ORM` combined with a generous amount of the `MVC` design pattern.
